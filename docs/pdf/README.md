# Entregables

## Ideacion.pdf

> La ideacion es un **proceso creativo** donde se produce un intercambio de ideas y opiniones para proponer alternativas de solucion al reto. Durante la ideacion se genera un importante vinculo con la imaginacion, por tanto, todas las Ideas son Validas. Para fortalecer la generacion de ideas, es importante la revicion previa de los recursos guia y las respuestas a la pregunta guia que cada equipo de trabajo realice.

## Solucion.pdf

> La etapa de solucion es aquella donde el equipo seleccion colaborativamente la idea con la que resolveran el reto, teniendo en cuenta que esta solucion debera ser concreta, articulada y factible de implementarse

## Prototipo

> El prototipo materializa la idea seleccionada para resolver el reto, pues se trata de la primera version que ayudara a comprender como funciona la propuesta para la solucion definitiva. El prototipo debe pasar por la etapa de validacion.
